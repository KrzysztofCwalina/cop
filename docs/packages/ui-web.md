# Web UI Package Reference

The `ui-web` package provides guidance for web UI development including accessibility, responsive design, and component patterns.

**Source:** [`packages/ui-web/`](../../packages/ui-web/)

## Import

```ruby
import ui-web
```

## Content

This is a guidance package that provides web development instructions for coding agents. It covers:

- Web accessibility standards
- Responsive design patterns
- Component architecture
- Web performance best practices
